# 24-Javascript

Small Typescript practice problems.

## Files

- `Problem-1.ts` - Check if either number is 100 or if their sum is 100.
- `Problem-2.ts` - Get a file extension from a filename.
- `Problem-3.ts` - Replace each character with the next alphabet letter.
- `Problem-4.ts` - Format and return the current date.
- `Problem-5.ts` - Append "New !" unless the word already includes "New".
- `Problem-6.ts` - Return the first and last three characters if length > 3.
- `Problem-7.ts` - Return the first half of an even-length word.
- `Problem-8.ts` - Merge two words without their first characters.
