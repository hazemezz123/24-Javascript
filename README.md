# 24-TypeScript

Small Typescript practice problems.

## Files

- `Problem-1.ts` - Check if either number is 100 or if their sum is 100.
- `Problem-2.ts` - Extract the file extension from a filename string.
- `Problem-3.ts` - Shift each character to the next letter in the alphabet.
- `Problem-4.ts` - Format and return the current date as a string.
- `Problem-5.ts` - Add "New !" to the start unless it already exists.
- `Problem-6.ts` - Return first/last three characters when length > 3.
- `Problem-7.ts` - Return the first half of an even-length word.
- `Problem-8.ts` - Merge two words while dropping their first characters.
- `Problem-9.ts` - Compare two values and return the one closest to 100, or a tie note.
- `Problem-10.ts` - Build a frequency map and check if a character repeats 2-4 times.
- `Problem-11.ts` - Separate even values and demonstrate filtering an array of numbers.
- `Problem-12.ts` - Count how many even numbers exist in an array and return the count text.
